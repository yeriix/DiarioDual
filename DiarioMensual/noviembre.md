### Noviembre 2025
 
Noviembre fue un mes intenso y variado, marcado por la gestión simultánea de múltiples proyectos web. Las tareas abarcaron desde mantenimiento y corrección de errores en webs existentes, hasta el desarrollo de nuevas funcionalidades, la creación de webs desde cero y la gestión de certificados, usuarios y contenidos multimedia.
 
**Mantenimiento y corrección de webs**
 
Una parte importante del mes se dedicó a corregir y pulir webs ya existentes. Se revisaron y repararon enlaces rotos en diversas páginas, se corrigió el footer de sostenibilidad en las páginas activas, y se resolvieron problemas de CSS en la web de IPRRS, incluyendo la corrección de puntos y errores de estilos visuales. También se corrigieron emoticonos mal renderizados y faltas de ortografía en la web de ICCP.
 
Se trabajó en la corrección del banner de SEEIUC y se ajustó el ancho de ciertos elementos para mejorar la maquetación. Además, se revisó y corrigió la web de Inflamaderm S3 y se añadió el código de conducta a la sección de Dermatología.
 
**Gestión de contenido e imágenes**
 
Se realizaron varios cambios de imágenes en distintas webs: se actualizaron las fotos de ponentes en la web del monográfico `bocemtium.com` mediante shortcodes, se cambió la imagen principal de IPRRS, y se corrigieron las fotos del slider de IPRRS. También se añadió una portada de directo en Gasull y un nuevo slider en Silauhe dentro de la sección de excelencia.
 
**Páginas nuevas y desarrollo de contenido**
 
Se crearon y completaron varias páginas nuevas durante el mes:
 
- **Página de privacidad de IPRRS**: se finalizó su desarrollo.
- **Página de Dermatoscopía**: se desarrolló desde cero dentro del entorno de Inflamaderm.
- **Web de SEEGG**: se continuó y se terminaron sus páginas principales.
- **Web de CUN**: se inició su desarrollo.
- **Web de ADO**: se creó desde cero, incluyendo la creación de usuarios para dicha plataforma.
- **Web de Pichia**: se desarrolló la sección de sponsors, se añadieron logos y se integró una newsletter.
- **Web de Mobility**: se inició su desarrollo, se corrigió el banner y se integraron ajustes con WPML para gestión multiidioma. También se rehízo la sección de certificados.
**IPRRS Multiidioma**
 
Se trabajó de forma recurrente y progresiva en la internacionalización de IPRRS, implementando soporte multiidioma. Esto incluyó la configuración de botones de idioma, la adaptación de contenidos y la gestión de traducciones. Se iteró varias veces a lo largo del mes para afinar el funcionamiento.
 
**Certificados y usuarios**
 
La gestión de certificados fue una tarea transversal durante noviembre. Se trabajó en los certificados de otoño (creación, configuración y gestión de cuestionarios y usuarios asociados), los certificados de Inflamaderm, los certificados de UiTB (creación y envío de resultados) y los certificados de Mobility (rehaciendo el formato adaptado a la nueva web). También se modificó y actualizó el CSV de usuarios, se importaron nuevos registros para la reunión de otoño, y se crearon usuarios nuevos en Bocemtium.
 
**Funcionalidades y desarrollo técnico**
 
Se implementaron varias funcionalidades nuevas de cierta complejidad técnica. Se finalizó el desarrollo de un **shortcode personalizado** y se realizaron sus traducciones. Se creó el sistema de gestión de **sesiones e inscripciones virtuales para Infladerm**. Se desarrolló un **modal con checkbox** que se activa al hacer clic, con configuración de botones y lógica asociada. Se implementaron **rutas relativas en Techaiderm** para mejorar la compatibilidad. Se configuraron herramientas de **analítica web para Jannu Ferides**. Se creó el **banner principal de SEEIUC** en formato grande, y se trabajó en la configuración del sistema de **correos automatizados de pedidos para Jannu Ferides**.
 
**Hardware y soporte**
 
Se configuraron dos portátiles de Amara y Susana, se tomaron fotos del soporte para la tablet, y se montó y desmontó el soporte físico de tablet para uso en eventos. También se configuró una tablet para el grupo diferido de Inflamaderm y Gasull.
 
**Vídeos y multimedia**
 
Se subieron vídeos al servidor S3, se descargó contenido de Vimeo y se gestionaron vídeos relacionados con Cancer Mama. También se añadió un iframe de directo en Gasull para retransmisión en tiempo real.
 
**Comunicaciones y correos**
 
Se enviaron correos a la secretaría de SEEIUC y se prepararon los envíos de comunicaciones de SEEGG. También se trabajó en la lógica de envío de resultados por correo electrónico.
